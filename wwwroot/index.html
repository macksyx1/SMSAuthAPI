📘 SMS Auth API Documentation

This API provides endpoints to send and verify SMS-based verification codes using Twilio. You can test them here or in Postman using the details below.

POST /api/verification/send

Sends a verification code to a specified phone number.

🔹 Headers (for Postman)
Content-Type: application/json
🔹 Request Body (JSON)
"+1234567890"
🔹 Example Curl Command
curl -X POST https://smsauthapi.onrender.com/api/verification/send \
  -H "Content-Type: application/json" \
  -d "\"+1234567890\""
🔹 Example Response
{
  "status": "pending"
}

POST /api/verification/check

        
Checks whether a verification code is valid for a given phone number.


        
🔹 Headers (for Postman)

        
Content-Type: application/json
    


        
🔹 Request Body (JSON)

        
{
  "phoneNumber": "+1234567890",
  "code": "123456"
}
    


        
🔹 Example Curl Command

        
curl -X POST https://smsauthapi.onrender.com/api/verification/check \
  -H "Content-Type: application/json" \
  -d "{ \"phoneNumber\": \"+1234567890\", \"code\": \"123456\" }"
    


        
🔹 Example Response

        
{
  "status": "approved"
}
