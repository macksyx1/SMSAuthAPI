📘 SMS Auth API Documentation for CarWash App

This API provides SMS-based verification functionality using Twilio. You can use the endpoints below to send and verify OTP codes.

POST /api/verification/send

Send a verification code to a phone number.

Request Body (JSON)

{
  "phoneNumber": "+1234567890"
}

POST /api/verification/check

        
Verify a code that was sent to a phone number.


        
Request Body (JSON)

        
{
  "phoneNumber": "+1234567890",
  "code": "123456"
}
